写在最前面

发表于 2018-12-26 置顶 | | 分类于 Greetings

起因：

  1. 一个机缘巧合，于一年前的某份周报中添加了【特别款模块】。这个模块记录了:
      a.这周发现的有趣新闻
      b.有趣的开源项目
      c.一些观点和随想
      d.一些资料和工具
      f.自己在业余做的点小玩意儿

      之后收到许多同事的回复，觉得这个模块很有趣，表示每周一都十分期待地打开邮箱，认真查阅邮件。这件事使我非常开心。

      之后的一段时间里，有些同事离开，有些同事加入，为了让大家都能看见【特别款模块】，于是建了blog把这些分享出来。

  2. 我的文笔实在是很糟糕，也许可以借着这个改善一下。

  3. 很早以前也发过一些东西，但之后，再回头看，发现当时的观点、眼界十分渺小，每次看完都想删掉。不如发出来，经常提醒自己是渺小且短暂来的实际，免得自己自夸。

  4. 我有个喜好，就是喜欢看旧闻，特别是十年前预测未来的新闻，什么经济走向、科技走向，世界形势等等，然后就可以对照当下的形势，有些观点很有远见，百来年前的预见果然实现了，有些观点纯属扯淡，与现实背道而驰。这些都很有意思，缺个地方把这些串联起来。

  也许以后还会有许多这样那样的原因使我继续做这个事儿。不管怎么说，Nathan餐馆算是开张了，借用深夜食堂里的开头：

  一日が终わり、人々が家路へにところ、俺の一日が始まる。

メーニュはこれだけ、あとは胜手にしてくれるや、できるもんが作るよってのが、俺の営业方针さ。 営业时间が夜12时から朝7时まで、人は深夜食堂っていってるんや。客が来るかって？其れがけっこう来るんだよ。

人们结束一天的忙碌工作，正赶着回家之际，我的一天才刚刚开始。
菜单祇有这些，随客人心意下单，祇要能煮得到的都会做，这是我的经营理念。
营业时间是午夜十二时至凌晨七时，大家都叫这里做深夜食堂。
问我有没有客人，还是挺多的。

Maltesers

Davis
